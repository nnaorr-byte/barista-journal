# ☕ יומן בריסטה חכם — Barista Journal AI

אפליקציית PWA בעברית לתיעוד, למידה וכיול של אספרסו ביתי, מותאמת ל-DeLonghi EC685 עם פורטפילטר Bottomless.

## הרצה

```bash
npm install
npm run dev      # שרת פיתוח על http://localhost:5173
npm run build    # בניית גרסת ייצור לתיקיית dist
```

בדפדפן בטלפון אפשר להתקין את האפליקציה למסך הבית (Add to Home Screen) — היא עובדת גם אופליין.

## מה יש בפנים

- **מסך בית** — המלצת השוט הבא (עם ציון ביטחון), סטטיסטיקה אישית, השוט הטוב ביותר, תזכורות תחזוקה.
- **שוט חדש** — זרימה בארבעה שלבים: הגדרה ← המלצה + טיימר ← תוצאות ← ניתוח AI Coach.
- **AI Coach** — אבחון Under/Over Extraction/Channeling, המשתנה היחיד לשינוי והסבר למה.
- **Dial-In Session** — שקית חדשה פותחת סשן כיול שמוודא שינוי משתנה אחד בלבד בין שוטים.
- **ניהול פולים** — פולים ושקיות, מעקב טריות (ימים מקלייה), מלאי ועלות לשוט.
- **Dashboard** — גרפים של דירוג/Yield/זמן/טחינה, השוואת פולים וקליות, פרופיל טעם, Top 10.
- **ייצוא** — CSV, Excel, גיבוי JSON מלא ושחזור.

## ארכיטקטורה

```
src/
├── domain/types.ts        # ישויות: Shot, Bean, Bag, Machine, Grinder, DialInSession…
├── db/
│   ├── database.ts        # Dexie (IndexedDB) + זריעת ציוד ראשונית
│   └── repositories.ts    # שכבת גישה לנתונים (מוכנה להחלפה בסנכרון ענן)
├── services/
│   ├── recommendation.ts  # מנוע המלצות: חוקים ← טריות ← למידה אישית
│   ├── dialInCoach.ts     # AI Coach — אבחון וייעוץ אחרי כל שוט
│   ├── learning.ts        # למידת העדפות אישיות (ממוצע משוקלל לפי דירוג)
│   ├── stats.ts           # חישובי Dashboard, מגמות, צריכה ועלות
│   ├── maintenance.ts     # כללי תחזוקה ל-EC685 ולמטחנות
│   └── importExport.ts    # CSV / Excel / גיבוי / שחזור
└── ui/                    # מסכים ורכיבים (React, RTL, ערכת קפה כהה)
```

כל שוט מקושר ל-`userId`/`machineId`/`grinderId` — הוספת מכונה, מטחנה או משתמש נוסף בעתיד אינה דורשת שינוי סכמה. מנוע ההמלצות בנוי כשרשרת מודולים עם נקודת חיבור מוכנה ל-Claude API.
